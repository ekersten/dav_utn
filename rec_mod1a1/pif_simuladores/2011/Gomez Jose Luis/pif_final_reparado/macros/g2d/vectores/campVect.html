Macro: RungeKutta
tipo 2D

Esta escena se usa para generar el macro.

Esta escena es una aplicación del macro.
El usuario puede definir como quiera el campo vectorial escribiendo las expresiones
para sus coordenadas Fx y Fy y pulsando intro.
dl es la separación vertical y horizontal entre dos flechas consecutivas.
normal significa que las flechas se toman de magnitud 1, si normal=no
entonces las flechas tienen la magnitud raíz(Fx^2+Fy^2).
factor se multiplica por la magnitud de las flechas para escalarlas.
