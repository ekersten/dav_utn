Macro: RungeKutta
tipo 2D

Esta escena se usa para generar el macro.
Como puede apreciarse, este macro no tiene parte gráfica, consta sólo de auxiliares
que realizan la integración de un sistema de ecuaciones diferenciales lineales por el
método de Runge-Kutta de orden 4.

Esta escena es una aplicación del macro que muestra la solución de un sistema
de ecuaciones diferenciales lineales de primer orden con la condición inicial
dada por el control gráfico. La solución se dibuja de [0,T] y dt es el avance en
cada paso de la integración numérica, que se hace con el método de Runge Kutta.
